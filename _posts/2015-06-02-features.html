Guidebook of User Scenarios with Screenshots

With Scenarioo you can document the important test scenarios of your application with UI screenshots and arbitrary additional information, generated out of your automated UI tests. Through the Scenarioo Docu Viewer web application anybody can jump, allmost like a kangaroo, through the steps and screenshots of the tested user scenarios of your application's use cases.

Accessible for Everybody

Everybody involved in the development of your software can use the Scenarioo Docu Viewer to browse comfortably inside the generated documentation. This is a common source of information and basis for collaboration for product owners, business analysts, product managers, requirements engineers, testers, supporters, developers and operators.

Always Up to Date

Scenarioo is a living documentation that is generated and updated every night, when you run your automated UI tests on your build server. Scenarioo turns your UI tests into a functional documentation of your software that is allways in synch with what is currently implemented.

UI Testing Toolkit Independent

Scenarioo provides a documentation format that is independent of your testing technology. It is possible to write Scenarioo XML files and screenshots from arbitrary UI testing toolkits and programming languages. For your convenience we provide a small scenarioo docu writer library that helps writing the documentation files in your tests for C#, Java and JavaScript.
